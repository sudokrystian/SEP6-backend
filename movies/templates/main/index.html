Welcome to the Movies API

Currently available endpoints:

    Movies:

  • Get movie by ID: /movies/{movie_id}
  • Get movie images: /movies/{movie_id}/images
  • Get movie crew by movie ID: /movies/{movie_id}/credits
  • Find similar movies: /movies/{movie_id}/similar
  • Last week tranding movies: /movies/trending
  • Find movies by name: /search/movie/page/{page_number}/name/{movie_name}

    • People:

  • Get person by ID: /people/{person_id}
  • Get people trending this week: people/trending
  • Find people by name: search/people/page/{page_number}/name/{person_name}
  • Find person credits: people/{person_id}/credits

    • Ratings:

  • Get all the ratings for the movie: rating/movies/{movie_id}
  • Get average rating of the movies the person starred in: /rating/person
  • Get the movie rating given by the logged in user: /rating/user/{movie_id}
  • Get all the ratings given away by the user: /rating/user
  • Put new rating to the movie: /rating

    • Movie lists

  • Find the lists for the user or create a new one: /list
  • Delete the list: /list/{list_id}
  • Find detailed lists for the user: /list/details
  • Find the movies for the specified list: /list/{list_id}/movies
  • Add movie to the existing list: /list/movies
  • Delete movie from an existing list: /list/{list_id}/movies/{movie_id}

    • Comments

  • Get comments for the movie or put a new one: /movies/{movie_id}/comments

    • User endpoints:

  • Crate a new user (register): /register
  • Obtain a token (login): /login
  • Refresh your token: login/refresh/
